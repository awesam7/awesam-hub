## About the Project

This project demonstrates a **personal resume/project page** hosted on an **AWS EC2 instance**. It dynamically shows:

- Visitor **location** based on IP
- Current **weather** using OpenWeatherMap API
- Current **time**

It also includes links to my:

- [GitHub Profile](https://github.com/awesam7)  
- [LinkedIn Profile](https://www.linkedin.com/in/sharma-sam)  
- **Live Weather Page** hosted on AWS EC2  

The page is built using **Flask** and styled with **HTML/CSS**, making it both interactive and visually appealing.  

## Features

- Dynamic **location detection** based on visitor IP  
- Real-time **weather and time display**  
- Buttons to navigate to **GitHub**, **LinkedIn**, and other project pages  
- Showcase of **DevOps & AI projects**  

## Tech Stack

- **Backend:** Python 3, Flask  
- **Frontend:** HTML, CSS  
- **Hosting:** AWS EC2 (t3.micro)  
- **APIs:** OpenWeatherMap, ip-api.com
